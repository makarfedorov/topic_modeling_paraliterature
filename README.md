# Тематическое моделирование жанров паралитературы
Репозиторий содержит материалы к тематическому моделированию текстов фанфикшн. В папке Corpus находится функция для скачивания текстов с метаинформацией с сайта ficbook, и сам скаченный корпус текстов (1600 единиц). Тамже находятся таблицы с количеством токнов для каждого из подкорпусов и ссылками, с помощью которых производилась скачивания текстов. В папке topic modeling находятся файлы с темами, которые выделились в результате моделирования и интерактивные визуализации. Файл Topic__Model_Fedorov содержит код и визуализации для исследования
